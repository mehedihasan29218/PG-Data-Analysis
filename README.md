# PG Data Analysis & Monitoring Dashboard

## 📌 Project Overview
This project is designed to monitor PG-connected telecom tower operations during power outages using interactive Power BI dashboards and analytical reporting. The solution helps track generator runtime, analyze outage trends, monitor operational costs, and evaluate vendor performance for better decision-making.

## 🚀 Key Features
- Hourly monitoring of PG-connected tower activity
- Power outage and generator runtime tracking
- PG operational cost monitoring and optimization
- Vendor performance and service efficiency analysis
- Historical trend analysis and forecasting
- KPI-driven management reporting
- Interactive dashboards with drill-down analysis

## 📊 Dashboard Insights
- Generator usage patterns during outages
- High-cost tower identification
- Vendor efficiency comparison
- Runtime optimization opportunities
- Historical PG usage trend analysis
- Operational continuity monitoring

## 🛠️ Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel
- Data Modeling

## 📈 Business Impact
This solution enables management teams to:
- Reduce operational expenses
- Improve vendor accountability
- Monitor generator performance efficiently
- Support faster data-driven decision-making
- Enhance operational visibility during outages
